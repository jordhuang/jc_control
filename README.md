# Jaynes--Cummings (JC) control 

This repository contains code to run the Jaynes--Cummings (JC) control package. 

JC control is a fast, gate-set-based protocol for the quantum control of an oscillator coupled to a qubit through a JC interaction. 

## Python requirements

The code is compatible with Python (3.13.11), PyTorch (2.10), QuTiP (5.2.3), NumPy (2.3.5), and h5py (3.15.1). It may work with later versions.
